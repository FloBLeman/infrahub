The Infrahub frontend UI displays the currently loaded [schema](../topics/schema.mdx) nodes in the top part of the left hand menu bar. This menu can be customized to fit the organizations needs and workflows

The showing and ordering of schema nodes can be controlled in two ways:

## Schema definition file

Options to nodes:

- `include_in_menu` - LINK to https://docs.infrahub.app/reference/schema/node
- `menu_placement` - LINK to https://docs.infrahub.app/reference/schema/node
